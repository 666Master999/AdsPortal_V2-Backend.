# Краткая инструкция для формы создания объявления (Vue)

---

## API
- POST `http://localhost:5131/api/ads`
- Content-Type: `multipart/form-data`
- Auth: `Authorization: Bearer {token}`

---

## Поля формы
- **Тип** (обязательно):
  - 0 = Продам
  - 1 = Куплю
  - 2 = Услуги
- **Заголовок** (обязательно): 3–20 символов
- **Описание** (необязательно): макс 3000 символов
- **Цена** (обязательно): 0–99999999999999
  - Если 0 → "Бесплатно"
  - Если "Договорная" (чекбокс) → поле цены скрыто, отправляется `isNegotiable: true`
- **Изображения** (необязательно): до 10 файлов
  - Первое изображение = главное
  - Можно менять порядок и выбирать главное

---

## Отправка
- Все поля отправляются через FormData
- Изображения добавлять как `images` (в нужном порядке)
- Если выбран "Договорная", отправить `isNegotiable: true` и цену = 0

---

## Пример запроса
```
title: "Продам iPhone"
type: 0
price: 50000
isNegotiable: false
description: "Отличное состояние"
images: [file1, file2, ...]
```

---

## Пример ответа
```json
{
  "id": 123,
  "title": "Продам iPhone",
  "type": 0,
  "price": 50000,
  "isNegotiable": false,
  "description": "Отличное состояние",
  "imageUrls": ["/files/1/userAds/123/1.jpeg", ...]
}
```

---

## Валидация
- Тип — обязательно
- Заголовок — 3–20 символов
- Цена — обязательно, если не "Договорная"
- Описание — макс 3000 символов
- Изображения — до 10 файлов
